
* 操作/動作

  1. [ ] dir
     - ダイアログ表示
     - 選択中dir用のラベルテキスト更新
     - genをenable

  2. [ ] month
     - 変更時の動作はとくになし(デフォルトコントロールで十分)

  3. [ ] gen
     - [ ] mediatorに通知
     - [ ] mediator
       - [ ] dirとmonthに応じて候補を生成
       - [ ] 対戦ツリーを生成
       - [ ] 最初の対戦を選択
         - [ ] 候補なし → パネルクリア
         - [ ] 1枚のみ → パネル片側のみ設定
         - [ ] 2枚 → ナエル両方設定

  4. [ ] prev
     - mediatorに通知，ひとつ前の対戦を選択

  5. [ ] next
     - mediatorに通知，現在の選択状態を保存して次の対戦を選択．選択処理はgenと同じ

  10. double-click panel
